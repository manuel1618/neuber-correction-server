Mathematical Foundation
1. Neuber's Rule

The product of stress and strain remains constant:

$$\sigma \cdot \varepsilon = \frac{\sigma_e^2}{E}$$
Where $\sigma$ = actual stress, $\varepsilon$ = actual strain, $\sigma_e$ = elastic stress, $E$ = Young's modulus
2. Ramberg-Osgood Material Model

Material stress-strain behavior:

$$\varepsilon = \varepsilon_e + \varepsilon_p = \frac{\sigma}{E} + \varepsilon_0 \left(\frac{\sigma}{\sigma_y}\right)^n$$
Where $\varepsilon_e$ = elastic strain, $\varepsilon_p$ = plastic strain, $\varepsilon_0$ = yield offset (0.002), $\sigma_y$ = yield strength, $n$ = hardening exponent
3. Hardening Exponent

Calculated from material properties:

$$n = \frac{\ln(\varepsilon_p^u) - \ln(\varepsilon_0)}{\ln(\sigma_u/\sigma_y)}$$
Where $\varepsilon_p^u = \varepsilon_u - \sigma_u/E$ = plastic strain at ultimate strength, $\sigma_u$ = ultimate tensile strength
Key Assumptions & Limitations
✓ When to Apply
  • Elastic stresses exceed yield strength
  • Fatigue analysis requiring accurate predictions
  • Materials with plastic deformation
  • FEA result correction
⚠ Limitations
  • Monotonic loading only (no cyclic)
  • Proportional loading assumption
  • Requires accurate material properties
  • May overcorrect below yield
ℹ Physical Interpretation
  1. Elastic Analysis: Linear behavior assumption, overestimates stress
  2. Plastic Correction: Accounts for yielding, reduces stress
  3. Intersection Point: Where Neuber's rule and material behavior meet
  4. Numerical Stability: Continuous solution across all stress ranges

Implementation Note

This calculator uses a continuous solution approach that may result in corrected stresses below the yield strength when the elastic stress is around or below yield. This is intentional for numerical stability and ensures smooth behavior across all stress ranges, making it suitable for analyzing FEA results that span both elastic and plastic regions.
